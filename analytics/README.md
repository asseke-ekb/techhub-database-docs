# TechHub - Аналитическая документация

## О системе

**TechHub** — цифровая платформа технопарка «Астана Хаб», обеспечивающая поддержку IT-экосистемы Казахстана.

### Ключевые метрики платформы

| Показатель | Значение |
|------------|----------|
| Зарегистрированных пользователей | 54,063 |
| Компаний | 6,639 |
| Заявок на услуги | 56,322 |
| Мероприятий | 1,459 |
| Вакансий | 1,205 |
| Бронирований | 8,628 |
| Таблиц в БД | 157 |

## Целевая аудитория платформы

### Пользователи

| Тип пользователя | Описание | Возможности |
|------------------|----------|-------------|
| Физическое лицо | Обычный пользователь | Просмотр событий, вакансий, подача заявок |
| IT-специалист | Пользователь с тегом `tag_it_specialist` | Отклик на вакансии, участие в TechOrda |
| Стажёр | Выпускник TechOrda (`tag_intern`) | Поиск работы, участие в мероприятиях |
| Инвестор | Пользователь с тегом `tag_investor` | Просмотр стартапов, инвестирование |
| Эксперт | Роль `expert` | Оценка заявок на услуги |

### Компании

| Тип компании | Тег | Описание |
|--------------|-----|----------|
| IT-компания | `tag_it_company` | Аккредитованная IT-компания с налоговыми льготами |
| Стартап | `tag_startup` | Компания в реестре стартапов РК |
| Резидент технопарка | `tag_techpark` | Резидент Astana Hub с доступом к офисам |
| Корпоративный партнёр | `tag_corp_partner` | Партнёр для корпоративных программ |
| НИИ | `tag_nii` | Научно-исследовательский институт |

## Основные бизнес-функции

### 1. Управление пользователями и компаниями
- Регистрация пользователей (email/телефон/eGov)
- Регистрация и верификация компаний
- Управление членством в компаниях
- Система тегов и статусов

### 2. Государственные услуги
- Аккредитация IT-компаний
- Включение в реестр стартапов
- Резидентство технопарка
- Гранты Seed Money
- Визовая поддержка

### 3. Контент-платформа
- Мероприятия (онлайн/офлайн/гибрид)
- Вакансии и отклики
- Блоги и статьи
- Новости экосистемы

### 4. Бронирование
- Переговорные комнаты
- Рабочие места
- Конференц-залы

### 5. Образование (TechOrda)
- Образовательные программы
- Заявки на обучение
- Отслеживание студентов

### 6. НИОКР
- Научно-исследовательские проекты
- Заявки недропользователей
- Экспертиза проектов

### Ключевые связи

```
ПОЛЬЗОВАТЕЛЬ (account_user)
    │
    ├── состоит в ──→ КОМПАНИЯ (account_company)
    │                     │
    │                     ├── подаёт ──→ ЗАЯВКА НА УСЛУГУ
    │                     ├── создаёт ──→ СОБЫТИЕ
    │                     └── размещает ──→ ВАКАНСИЯ
    │
    ├── подаёт ──→ ЗАЯВКА НА УСЛУГУ (service_servicerequest)
    ├── создаёт ──→ СОБЫТИЕ (core_event)
    ├── создаёт ──→ БЛОГ (core_blog)
    ├── бронирует ──→ БРОНИРОВАНИЕ (booking_booking)
    └── откликается ──→ ВАКАНСИЯ (core_vacancycandidate)
```
