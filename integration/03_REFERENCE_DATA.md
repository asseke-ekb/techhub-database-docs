# Справочные данные для интеграций

## 1. Роли пользователей

### Системные роли (account_user.role)

| Код | Название | Описание |
|-----|----------|----------|
| `user` | Пользователь | Обычный пользователь |
| `admin` | Администратор | Расширенные права |
| `superadmin` | Суперадмин | Полный доступ |
| `moderator` | Модератор | Модерация контента |
| `expert` | Эксперт | Оценка заявок |
| `manager` | Менеджер | Управление заявками |
| `cett_expert` | Эксперт ЦЭТТ | Экспертиза ЦЭТТ |
| `cett_moderator` | Модератор ЦЭТТ | Модерация ЦЭТТ |
| `techorda_manager` | Менеджер TechOrda | Управление TechOrda |
| `niokr_admin` | Админ НИОКР | Управление НИОКР |

### Роли в компании (account_usercompany.role)

| Код | Название | Описание |
|-----|----------|----------|
| `owner` | Владелец | Полные права, один на компанию |
| `member` | Сотрудник | Действия от имени компании |
| `advisor` | Консультант | Ограниченный доступ |

## 3. Статусы контента

### Статусы публикации (event, vacancy, blog)

| Код | Название | Описание |
|-----|----------|----------|
| `draft` | Черновик | Не отправлен на модерацию |
| `sent` | На модерации | Ожидает проверки |
| `correction` | На доработке | Требует исправления |
| `success` | Опубликован | Прошёл модерацию |
| `reject` | Отклонён | Отклонён модератором |
| `deleted` | Удалён | Удалён |
